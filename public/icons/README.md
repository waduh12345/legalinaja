## Desain Icon:

- **Warna utama**: Hijau (#059669) - warna Islam
- **Simbol**: Ka'bah, bulan sabit, atau tulisan Arab "الله"
- **Style**: Modern, minimalis, mudah dikenali
- **Format**: PNG dengan transparansi

## Tools untuk membuat icon:

1. **Online**: Canva, Figma, Adobe Express
2. **Desktop**: Adobe Illustrator, GIMP, Inkscape
3. **AI Generated**: DALL-E, Midjourney dengan prompt "Islamic app icon green color"

## Template Prompt untuk AI:

"Create a modern Islamic app icon with green color (#059669), featuring Ka'bah or crescent moon, minimalist design, 512x512 pixels, PNG format with transparency"

## Catatan:

- Semua icon harus memiliki background transparan
- Icon harus terlihat jelas di berbagai ukuran
- Konsisten dengan tema aplikasi ibadah
- Sesuai dengan guidelines PWA dari Google
